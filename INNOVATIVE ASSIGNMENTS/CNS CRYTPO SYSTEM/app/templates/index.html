How to Interpret This Graph

  • X-axis (Time): Each tick represents one simulation time step.
  • Y-axis (Throughput): Number of successful transactions per step — higher is better.
  • Stable high line: Network is efficient and stable.
  • Fluctuating or dropping line: Indicates congestion, attacks, or node failures.
